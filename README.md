# POS UI Extensions Tutorials

Example apps demonstrating [POS UI extensions](https://shopify.dev/docs/api/pos-ui-extensions) features and capabilities.

## Running an Example

Each example is a standalone app. To run an app locally, clone the repository and then follow our [Getting started](https://shopify.dev/docs/apps/build/pos/getting-started#step-2-run-the-local-extension-in-your-dev-store) tutorial to install the project dependencies and preview the app.

## Contributing
